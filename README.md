# LPサンプル集

LP（ランディングページ）作成の受注用に、様々なイベント・業種を想定した5種類のサンプルLPを作成しました。

## 概要

それぞれ全く異なるデザインパターンとターゲット層を想定し、HTML/CSS/JavaScriptで実装した静的サイトです。

## サンプルLP一覧

### 01. 音楽フェスティバルLP
**テーマ**: ビビッドでエネルギッシュなデザイン

**特徴**:
- 派手なグラデーション背景とアニメーション
- 高いエネルギー感を表現
- 若年層ターゲット
- チケット販売、ラインナップ紹介

**カラー**: ピンク、パープル、ブルーの鮮やかなグラデーション

**ディレクトリ**: `01_music_festival/`

---

## 使い方

各ディレクトリの `index.html` をブラウザで開くだけで、そのままLPとして閲覧できます。

```bash
# 例: 音楽フェスティバルLPを開く
open 01_music_festival/index.html
```

## 共通機能

全てのLPに以下の機能を実装しています：

- ✅ レスポンシブデザイン（スマートフォン対応）
- ✅ スムーズスクロール
- ✅ スクロールアニメーション
- ✅ インタラクティブな要素（ホバーエフェクト等）
- ✅ フォーム実装（デモ版）

## カスタマイズ

各LPは独立したファイル構成になっているため、以下のカスタマイズが容易です：

- **色の変更**: CSSファイルの色コードを変更
- **テキスト変更**: HTMLファイルの内容を編集
- **画像追加**: プレースホルダー部分に画像のURLを設定
- **レイアウト調整**: CSSのグリッドやフレックスボックスを調整

## 技術仕様

- **HTML5**: セマンティックなマークアップ
- **CSS3**: フレックスボックス、グリッドレイアウト、アニメーション
- **JavaScript (Vanilla)**: ライブラリ不使用で軽量
- **レスポンシブ**: モバイルファーストの設計
